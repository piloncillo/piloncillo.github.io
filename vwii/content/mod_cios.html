Instalar cIOS en el modo virtual Wii


Los custom IOS (cIOS) constituyen el CFW del modo virtual Wii. Los cIOS quedan instalados en la memoria interna de la Wii U (SLCCMPT) y no dependen de ninguna forma de la modificación del modo Wii U (principal) ni de archivos en la tarjeta SD después de ser instalados.

Instalar cualquier IOS o cIOS hecho para la Wii clásica en el modo virtual Wii causará un brick en el modo virtual Wii.
Asegúrate de no tener archivos .wad en ningún otro lugar que no sea la carpeta sd:/app.
  • En este punto debes tener la Wii U encendida en el modo virtual Wii con la tarjeta SD introducida. El Homebrew Channel debe estar en el menú de Wii.
  • Abre el Homebrew Channel.
  • Carga d2x cIOS Installer.
  • Presiona cualquier botón del Wii Remote para continuar.
  • Usa las flechas del D-Pad del Wii Remote para que coincida con lo siguiente:
    • Select cIOS: d2x-v11-beta3-vWii
    • Select cIOS base: 38
    • Select cIOS slot: 248
  • Presiona el botón A para instalar.
  • Espera a que el cIOS se instale.
  • Presiona el botón A para continuar.
  • Usa las flechas del D-Pad para que coincida con lo siguiente:
    • Select cIOS: d2x-v11-beta3-vWii
    • Select cIOS base: 56
    • Select cIOS slot: 249
  • Presiona el botón A para instalar.
  • Espera a que el cIOS se instale.
  • Presiona el botón A para continuar.
  • Usa las flechas del D-Pad para que coincida con lo siguiente:
    • Select cIOS: d2x-v11-beta3-vWii
    • Select cIOS base: 57
    • Select cIOS slot: 250
  • Presiona el botón A para instalar.
  • Espera a que el cIOS se instale.
  • Presiona el botón A para continuar.
  • Usa las flechas del D-Pad para que coincida con lo siguiente:
    • Select cIOS: d2x-v11-beta3-vWii
    • Select cIOS base: 58
    • Select cIOS slot: 251
  • Presiona el botón A para instalar.
  • Espera a que el cIOS se instale.
  • Presiona el botón B para salir.

Parchar el IOS 80

Si por alguna razón, la Wii U se apaga mientras se aplica el parche al IOS 80, el modo virtual Wii quedará bloqueado (brick). Esto se puede solucionar extrayendo el IOS 80 de la copia de seguridad la NAND realizada anteriormente y luego reemplazando los archivos vía de FTP o restaurando el modo virtual Wii por completo.
  • En este punto debes estar en el Homebrew Channel.
  • Carga Patched IOS80 Installer for vWii.
  • Lee la advertencia y espera 30 segundos.
  • Presiona cualquier botón para instalar.
  • Espera hasta que se muestre el mensaje IOS80 Installation is complete!.
  • Presiona cualquier botón para salir.
