• Si activaste el arranque automático del PayloadLoader: Enciende la consola y mantén presionado el botón X hasta que salga el Environment Loader.
  • Si tu consola tiene PayloadLoader de arranque manual: Enciende la consola, accede al menú de Wii U, abre Información sobre salud y seguridad y mantén presionado el botón X del GamePad para abrir el Environment Loader.
  • Navega por el menú usando las flechas del D-Pad del GamePad, selecciona tiramisu.
  • Presiona el botón Y para configurar Tiramisu como el entorno predeterminado.
  • Presiona el botón A para abrir Tiramisu.
  • Si aparece un mensaje de advertencia acerca del update folder quiere decir que tu consola no está bloqueando las actualizaciones del sistema apropiadamente.
  • Presiona el botón X para bloquear por completo las actualizaciones del sistema de la consola o presiona el botón B para no volver a ver el mensaje de advertencia.
  • Bloquear las actualizaciones del sistema de la consola no afecta poder instalar las actualizaciones de los juegos. Bloquear las actualizaciones no es obligatorio, pero se recomienda encarecidamente. Puedes desbloquear las actualizaciones del sistema de la consola en cualquier momento después de completar la instalación de Tiramisu. Consulta la información de aquí cuando quieras desbloquear las actualizaciones.
  • Ahora debes estar en el Boot Selector de Tiramisu, con la opción Wii U Menu seleccionada, presiona el botón Y para configurar dicha opción como predeterminada y luego presiona el botón A para abrir el menú de Wii U.
  • En este punto ya finalizaste la instalación de PayloadLoader, del Environment Loader y de Tiramisu, tu consola iniciará con Tiramisu de forma predeterminada.
  • Con el entorno de Tiramisu cargado el menú de Wii U no se ve diferente al menú de Wii U normal.
  • En la siguiente página vamos a añadir aplicaciones y módulos que agregan funcionalidad a Tiramisu.
  • Toma nota de los cuadros de información de abajo para usar correctamente Tiramisu.
Con el entorno de Tiramisu cargado en la consola debes abrir el Editor de Mii para acceder al Homebrew Launcher. El Homebrew Launcher del Editor de Mii está hecho para funcionar con Tiramisu, otros Homebrew Launcher no están garantizados para funcionar correctamente con Tiramisu, no se recomienda usarlos.
Para ir al Editor de Mii, simplemente presiona el botón HOME mientras estás en el Homebrew Launcher.
Si activaste el arranque automático del PayloadLoader puedes abrir el Environment Loader si enciendes la consola y mantienes presionado el botón X.
Si activaste el arranque automático del PayloadLoader puedes abrir el menú Boot Selector si enciendes la consola y mantienes presionado el botón START ⊕︀.
Si activaste el arranque automático del PayloadLoader puedes impedir que cargue el Environment Loader y Tiramisu sacando la tarjeta SD de la consola antes de encenderla. De esta forma la consola iniciará sin modificaciones. La siguiente vez que enciendas la consola con la tarjeta SD con los archivos del Environment Loader y Tiramisu cargará estos normalmente.
